# Memory Bank

Эта папка хранит актуальный контекст проекта для быстрого онбординга и стабильной разработки.

## Файлы
- `project-overview.md` — что это за проект и текущий статус
- `architecture.md` — как устроена система по слоям
- `decisions.md` — принятые технические решения
- `runbook.md` — команды запуска, тестов и отладки
- `backlog.md` — ближайшие улучшения

## Правило обновления
После заметных изменений кода обновляйте минимум:
1. `project-overview.md` (что поменялось)
2. `decisions.md` (если были новые решения/ограничения)
3. `runbook.md` (если менялись команды или требования)
