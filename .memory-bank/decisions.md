# Technical Decisions

## Принятые решения
1. UI на `PySide6`.
2. Основное окно максимально компактное, без вкладок.
3. Настройки вынесены в отдельный диалог, открываемый из трея.
4. Данные хранятся локально (`JSON + SQLite`), без облака.
5. Учет времени: только активное, idle исключается.
6. Двухуровневые напоминания: soft/hard.
7. Для hard-этапа используется fullscreen overlay.
8. Тикер — `QTimer` раз в секунду.

## Ограничения
- В текущем окружении (Astra Linux) чаще нужен `virtualenv`, а не `python -m venv`.
- Для Python 3.7 зафиксирован `PySide6<6.6`.
- Кнопка мягкого напоминания закрывает уведомление (не запускает break напрямую).
