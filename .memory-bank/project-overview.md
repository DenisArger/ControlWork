# Project Overview

## Назначение
ControlWork напоминает о перерывах и считает только активное время работы за компьютером.

## Текущий UX
- Компактное плавающее окно (always on top):
  - текущее состояние
  - время работы текущего цикла
  - время до следующего перерыва
- Основное управление через контекстное меню иконки в трее.
- Настройки открываются отдельным диалогом из трея.

## Основные сценарии
1. Пользователь запускает приложение.
2. Приложение в фоне считает активное время.
3. На soft-точках показывает мягкое напоминание.
4. На hard-точке показывает строгий экран перерыва.
5. После валидного перерыва цикл напоминаний сбрасывается.

## Хранилище
- `settings.json` для пользовательских настроек.
- `controlwork.db` (SQLite) для сессий/событий.

## Поддерживаемые платформы (MVP)
- Linux
- Windows
